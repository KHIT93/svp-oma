class control_logs.models.control_log.ControlLog(*args, **kwargs)[source]¶

Bases: django.db.models.base.Model

The ControlLog class is responsible for logs about a specific WindTurbine.

Each entry holds the following information:

Parameters:
  • id – Unique identifier for the logs of the WindTurbine
  • timestamp – Timestamp for the entry in the log
  • sensor_type_data – Which kind of sensor, there is logged
  • current_value – The current value for the specific sensor
  • windmill_state – The state the WindTurbine was in
