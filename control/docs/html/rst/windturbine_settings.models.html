class windturbine_settings.models.windturbine_setting.WindturbineSetting(*args, **kwargs)[source]¶

Bases: django.db.models.base.Model

The WindturbineSetting class is responsible for holding the settings related to a specific WindTurbine

Each entry holds the following information:

Parameters:
  • id – Unique identifier for the settings of the WindTurbine
  • windturbine – Relationship to the WindTurbine using these settings. In the SQL database, this reference is saved as windturbine_id
  • state – The normal state for of operation the this WindTurbine
  • brake – Defines if the brake are active
  • wing_angle – Defines the angle of the wing
  • max_rpm_generator – The maximum threshold for the rpm at the generator
  • max_temp_gearbox – The maximum threshold for the temp of the gearbox
  • max_temp_generator – The maximum threshold for the temp at the generator
